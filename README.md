JavaScript Assignment - IGIRUBUNTU Fraternite (25RP01890)

A simple interactive web page built with **HTML, CSS, and JavaScript**.  
This project includes dynamic DOM manipulation, event handling, a simple calculator, and a to-do list.

---


## About

This project is a practice assignment for JavaScript.  
It demonstrates:

1. Changing the heading text dynamically  
2. Displaying messages on button click  
3. A simple calculator performing basic arithmetic  
4. A functional to-do list  

---

## Features

- Dynamic heading change with button click  
- Interactive message display on click  
- Simple calculator supporting `+`, `-`, `*`, `/`  
- Add, display, and manage tasks in a to-do list  
- Styled with CSS for a clean look
